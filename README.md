# 42_ft_ping
This project is about recoding the ping command

## Usage

### Compilation

```bash
make
```

### Execute

```bash
./ft_ping [args ...]
```

```
Usage
ft_ping [options] <destination>

Options:
        <destination>   dns name or ip address
        -v              verbose output
        -h              help
        -c              count <integer>
        -t              time to live <integer>
        -W              timeout <integer in second>
        -w              deadline <integer in second>
        -q              quiet
        -4              ipv4 only
```
